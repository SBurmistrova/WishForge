# WishForge

WishForge is a project-oriented platform that helps transform ideas and wishes into achievable outcomes.

Instead of keeping a static wishlist, users select a wish, turn it into a project, break it down into smaller steps, and gradually move toward completion.

## Features

* Convert wishes into structured projects
* Break goals into manageable steps
* Track progress automatically
* Focus on execution instead of endless planning
* Preserve completed journeys and reflect on achievements

## Tech Purpose

Developed as a backend portfolio project using Go to practice:

* API design
* Business logic implementation
* State management
* Project decomposition
* Backend architecture
